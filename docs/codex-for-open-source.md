# Codex for Open Source

## Why this repository is a fit

`maintainer-kit` is a public, MIT-licensed GitHub Action focused on human-in-the-loop maintainer
workflows. It is dogfooded in this repository and keeps mutating behavior behind explicit
maintainer approval.

## Public demo evidence

This repository includes a public dogfood demo under [docs/demo.md](demo.md).

The demo includes screenshots of:

- an Issue Intake Brief generated in this repository
- a PR Decision Brief generated in this repository
- the GitHub Actions workflow run used to produce the brief

This gives Codex for Open Source reviewers a quick way to verify that the project is not just a concept. It is already being used to improve its own maintainer workflow.
